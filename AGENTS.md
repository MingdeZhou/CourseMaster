# AGENTS.md

## 仓库协作约定

本仓库的开发代理在每次修改代码时，都必须同时维护开发文档。

## 强制要求

1. 每次功能开发、交互修改、结构调整、文案变化完成后，都要同步更新 [development/README.md](/c:/Users/lenovo/Desktop/CourseMaster/development/README.md)。
2. 文档统一使用中文书写。
3. 如果当前实现与 [local_vault_demo_tech_spec.md](/c:/Users/lenovo/Desktop/CourseMaster/local_vault_demo_tech_spec.md) 有偏移，必须在开发文档中明确记录。
4. 如果只是临时占位，也要写清楚哪些功能只是界面占位，还没有真正接通。

## 每次更新文档时至少要写的内容

- 本次改了什么
- 当前实现到了哪个 milestone
- 相对于基线文档有哪些变化
- 哪些部分已经能用
- 哪些部分仍未实现或只是占位

## 文档风格

- 使用中文
- 直接、清楚、可追踪
- 优先写事实，不写空话
- 面向后续继续开发的人，保证别人接手时能快速理解当前状态
